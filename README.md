# Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a sample dataset to
understand data structure, patterns, and basic statistics.

## Files
- `data.csv` – Dataset used for analysis
- `eda.py` – Python script for EDA
- `screenshots` branch – Contains output visualizations and results

## Key Analysis
- Data structure and types
- Summary statistics
- Distribution analysis
- Visual insights using plots

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

Screenshots of outputs are available in the `screenshots` branch.
